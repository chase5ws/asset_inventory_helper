### 專案說明

這是一個基於 **Python** 開發的 Excel 資訊資產盤點輔助工具，旨在簡化重複性操作並提升工作效率。透過此工具，使用者可以快速提取資料、生成下拉式選單、處理目標檔案，並自動生成結果報表。
